Typography: css styling for text in react-friendly way
-> Define in theme/typography


{ lg: 4, md: 6, sm: 6, xs: 12 } -> It defines how much horizontal space (columns) a component takes up on different screen sizes.
xs	Extra-small	Mobile (<600px)
sm	Small	Tablet (≥600px)
md	Medium	Small laptop (≥900px)
lg	Large	Desktop (≥1200px)
xl	Extra large	Very wide screen (≥1536px)

GRID Behavior:
-> horizontal by default 

<Grid container spacing={2}>
  <Grid item xs={4}>A</Grid>
  <Grid item xs={4}>B</Grid>
  <Grid item xs={4}>C</Grid>
</Grid> -> [A] [B] [C]
